This repo downloads nix
then downloads all the nix packages needed to run hashirama in a nixos VM
It can also deploy a hashirama nixos iso to a nvidia jetson or to a cloud server
It covers self hosted, on prem, cloud, iot and edge installations
To run.. nix build -> output a VM with woking
